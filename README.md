Just terminal exit codes with more description to use in dart

## Getting started

```dart
dart pub add all_exit_codes
```
## Usage

```dart
import 'package:all_exit_codes/all_exit_codes.dart';

Never main() {
  print('Maybe you are using this command wrong. Check the usage.');
  exit(wrongUsage);
}
```
